# CWRU 12k Drive End Raw Files

This directory is reserved for manually placed CWRU 12k Drive End bearing fault `.mat` files and Normal Baseline `.mat` files.

For the minimal 3 hp to 2 hp reproduction setup, place the required 20 `.mat` files here after downloading them manually from the CWRU Bearing Data Center.

Expected filename examples:

- `Normal_2.mat`
- `IR007_2.mat`
- `OR007@6_2.mat`

Do not commit raw `.mat` data files to git.
